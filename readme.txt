=== Formidable “Just Updated” Trigger ===
Requires at least: 4.0 ( Formidable Pro 2.01.02 or higher is required )
Tested up to: 4.9
Stable tag: 1.1

This is an add-on for Formidable Forms which allows you to trigger a form action only when the specified fields were *just* updated in the current form submit.
